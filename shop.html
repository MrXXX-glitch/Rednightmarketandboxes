Nmap — Network scanner

Nmap is an open-source tool for network discovery and security auditing. Use it to map networks, discover hosts and services, and perform port scanning.
  • Install: https://nmap.org
  • Quick example: nmap -sC -sV -oN scan.txt target.tld
  • Use-case: network inventory, service detection, CTF recon.
Official site

Burp Suite — Web proxy

Burp Suite is a web security testing platform (proxy, scanner, intruder). The Community Edition is free and great for manual testing; paid Professional adds automation and scanning features.
  • Install / download: https://portswigger.net/burp
  • Quick tip: set browser proxy to 127.0.0.1:8080 and install Burp CA cert to intercept HTTPS.
  • Use-case: intercepting requests, testing input validation, discovering vulnerabilities in web apps.
Burp Suite

Hydra (THC-Hydra) — Password cracking

Hydra is a fast password-cracking tool supporting many protocols (ssh, ftp, http-form, smb, etc.). Always use Hydra only against systems you have explicit permission to test.
  • Repo / docs: https://github.com/vanhauser-thc/thc-hydra
  • Example: hydra -l admin -P passwords.txt ssh://192.168.1.10
  • Use-case: auditing weak credentials in lab environments.
GitHub

Gobuster — Directory & vhost brute-forcing

Gobuster is a fast tool written in Go for brute-forcing URIs (directories/files), DNS subdomains, and virtual hosts using wordlists. It’s ideal for web recon in CTFs and pentest labs.
  • Install (Kali/Debian): sudo apt install gobuster or go install github.com/OJ/gobuster/v3@latest
  • Example: gobuster dir -u https://target.tld -w /usr/share/seclists/Discovery/Web-Content/raft-large-directories.txt -t 50
  • Use-case: find hidden directories, files, and vhosts on web servers.
GitHub

RustScan — fast port scanner

RustScan is a high-speed port scanner written in Rust that pipes results into Nmap for deeper service/version detection. It’s great when you need a quick host sweep before detailed scanning.
  • Install: curl -sS https://raw.githubusercontent.com/RustScan/RustScan/master/install.sh | sudo bash
  • Example: rustscan -a 10.10.10.10 -b 2000 -- -A -sV -oN rustscan-nmap.txt
  • Use-case: fast port discovery, then hand off to Nmap for details.
GitHub

John the Ripper — Password cracker

John is a flexible password cracking tool that supports many hash formats. Use it for offline password analysis on captured hash files (e.g., from /etc/shadow or hash dumps).
  • Install: sudo apt install john or use the Jumbo build for more formats.
  • Example: john --wordlist=passwords.txt --format=sha512crypt hashes.txt
  • Use-case: recover weak passwords from hash dumps in a controlled lab.
Project site

Wireshark — Packet analyzer

Wireshark is a graphical network protocol analyzer for capturing and interactively browsing packet data. Use it to inspect protocols, debug issues, and learn how network traffic looks.
  • Download: https://www.wireshark.org
  • Quick tips: use capture filters (e.g., host 10.0.0.5) and display filters (e.g., http && tcp.port==80) to focus results.
  • Use-case: protocol debugging, forensic traffic analysis, CTF network challenges.
